# Navigation

Wayd's interface is organized into several key areas accessible from the sidebar navigation.

## Sidebar Navigation

The left sidebar provides access to all major areas of the application:

- **Organizations** - Teams and organizational structure
- **Planning** - Planning intervals, sprints, and roadmaps
- **Work Management** - Workspaces and work items
- **PPM** - Projects, portfolios, and strategic initiatives
- **Settings** - Application configuration and administration

## Header

The top header provides:

- **Search** - Global search (Ctrl+K) across Wayd and documentation
- **Notifications** - Activity notifications
- **Profile** - Account settings and sign out

## Global Search

Press **Ctrl+K** (or **Cmd+K** on Mac) to open the global search modal. Search results are grouped by category and scoped by your permissions.

### Searchable Entities

| Area | Searchable Items |
| --- | --- |
| **Work** | Work items, workspaces |
| **Organizations** | Teams, teams of teams, employees |
| **Planning** | Planning intervals, PI iterations, sprints, PI teams, PI objectives, roadmaps |
| **PPM** | Projects, programs, portfolios, strategic initiatives |

### Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| **Ctrl+K** / **Cmd+K** | Open search (App scope) |
| **Ctrl+D** | Switch to Docs scope (when open) |
| **↑ / ↓** | Navigate results |
| **Enter** | Open selected result |
| **Esc** | Close search |

## Breadcrumbs

Breadcrumb navigation appears below the header, showing your current location and allowing quick navigation to parent pages.
