# Project Portfolios

The Project Portfolio Management domain provides tools for managing projects and organizing them into portfolios.

## Key Concepts

### Projects

Projects represent distinct initiatives with defined scope, timeline, and deliverables. They can be:
- Tracked through lifecycle phases
- Associated with teams and work items
- Organized into portfolios

### Portfolios

Portfolios are collections of related projects that are managed together for strategic alignment and resource optimization.

### Portfolio Status

Portfolios and projects have status tracking that provides visibility into health and progress.

## Common Tasks

- Creating and managing projects
- Organizing projects into portfolios
- Tracking project status and health
- Viewing portfolio-level dashboards
