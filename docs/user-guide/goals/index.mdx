# Goals

The Goals domain provides an OKR (Objectives and Key Results) framework for tracking strategic and team-level goals.

## Key Concepts

### Objectives

Objectives are qualitative goals that describe what you want to achieve. They are:
- Time-bound (associated with a planning interval)
- Owned by a team or individual
- Measurable through key results

### Key Results

Key results are quantitative measures that track progress toward an objective. They define specific, measurable outcomes that indicate whether the objective is being met.

## Common Tasks

- Creating objectives for a team
- Defining key results
- Tracking progress
- Linking objectives to work items
