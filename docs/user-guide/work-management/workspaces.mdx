# Workspaces

Workspaces are containers that organize work items and associate them with a work process.

## What is a Workspace?

A workspace provides:
- A scoped container for related work items
- Association with a specific work process (defining available work types and workflows)
- Ownership designation (Owned by Moda or Managed by an external system)

## Types of Workspaces

- **Owned Workspaces** - Created and managed within Moda
- **Managed Workspaces** - Synchronized from external systems (e.g., Azure DevOps projects)

## Creating a Workspace

1. Navigate to **Work > Workspaces**
2. Click **New Workspace**
3. Configure:
   - **Name** - Descriptive name for the workspace
   - **Work Process** - Select the work process to use
   - **Description** - Optional context about the workspace
4. Click **Save**
