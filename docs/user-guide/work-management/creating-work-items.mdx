# Creating Work Items

## Creating a New Work Item

1. Navigate to the workspace where you want to create the work item
2. Click **New Work Item**
3. Select the **Work Type** (e.g., User Story, Task, Bug)
4. Fill in the required fields:
   - **Title** - A clear, concise description
   - **Status** - Initial workflow state
5. Optionally set:
   - **Priority** - Relative importance
   - **Assignee** - Person responsible
   - **Parent** - Link to a parent work item
   - **Description** - Detailed information
6. Click **Save**

## Work Item Fields

| Field | Required | Description |
|-------|----------|-------------|
| Title | Yes | Brief description of the work |
| Type | Yes | Category defined by work process |
| Status | Yes | Current workflow state |
| Priority | No | Relative importance |
| Assignee | No | Responsible person |
| Parent | No | Hierarchical parent work item |
| Description | No | Detailed information and context |
