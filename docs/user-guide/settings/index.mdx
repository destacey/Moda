# Settings

The Settings area provides administrative tools for configuring Moda.

## User Management

Manage users, roles, and permissions:
- **Users** - Create, activate, deactivate, and manage user accounts
- **Roles** - Define permission groups
- **Login Providers** - Microsoft Entra ID or Local authentication

## Work Management Settings

- **Work Processes** - Define work types, workflows, and hierarchies
- **Work Statuses** - Configure available statuses and categories

## Feature Management

Control feature visibility using feature flags:
- Enable or disable features across the application
- Features are defined in code and managed through the UI
- See [Feature Flags](../../reference/feature-flags.mdx) for details

## Integrations

Configure connections to external systems:
- **Azure DevOps** - Work item synchronization
- **Microsoft Graph** - Employee/user synchronization
