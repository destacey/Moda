# Planning

The Planning domain provides tools for organizing work into time-boxed intervals and iterations.

## Key Concepts

### Planning Intervals

Planning intervals (PIs) are fixed time periods used to plan and track work across teams. They provide:
- A defined start and end date
- A collection of iterations within the interval
- Cross-team visibility into planned work

### Iterations

Iterations are shorter time-boxed periods within a planning interval (typically 2-4 weeks). Teams plan and commit to delivering work within each iteration.

## Common Tasks

- Creating and managing planning intervals
- Defining iterations within a planning interval
- Assigning work to iterations
- Tracking iteration progress
