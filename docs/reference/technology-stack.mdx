# Technology Stack

## Backend (.NET)

| Technology | Purpose |
|-----------|---------|
| .NET 10.0 | Runtime and SDK |
| ASP.NET Core | Web API framework |
| Entity Framework Core | ORM and database access |
| SQL Server | Database |
| MediatR | CQRS command/query dispatch |
| FluentValidation | Input validation |
| Mapster | Object mapping (DTOs) |
| NodaTime | Date/time handling |
| CSharpFunctionalExtensions | Result pattern |
| Hangfire | Background job processing |
| Serilog | Structured logging |
| NSwag | OpenAPI spec and client generation |
| Microsoft.FeatureManagement | Feature flags |
| ASP.NET Core Identity | User management |
| Azure AD / Local JWT | Authentication |
| OpenTelemetry | Observability |

| react-markdown | Markdown rendering |
| dayjs | Date formatting |
