# Prerequisites

Before setting up Wayd for local development, ensure you have the following installed.

## Required Software

| Software | Version | Purpose |
|----------|---------|---------|
| .NET SDK | 10.0+ | Backend API and services |
| Node.js | 22+ (LTS) | Frontend build and development |
| SQL Server | 2019+ or LocalDB | Database |
| Git | Latest | Source control |

## Optional but Recommended

| Software | Version | Purpose |
|----------|---------|---------|
| Docker Desktop | Latest | Containerized development |
| .NET Aspire | Latest | Local orchestration and telemetry |
| Visual Studio Code | Latest | Recommended IDE |
| Visual Studio | 2022+ | Alternative IDE for .NET |

## VS Code Extensions

- C# Dev Kit
- ESLint
- Prettier
- Thunder Client (API testing)
