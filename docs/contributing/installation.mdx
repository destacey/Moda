# Installation

## Clone the Repository

```bash
git clone https://github.com/destacey/Wayd.git
cd Wayd
```

## Build the Backend

```bash
dotnet build Wayd.slnx
```

## Install Frontend Dependencies

```bash
cd Wayd.Web/src/wayd.web.reactclient
npm install
```

## Set Up the Database

Wayd uses SQL Server. You can use SQL Server LocalDB, a full SQL Server instance, or a Docker container.

### Apply Migrations

From the repository root:

```bash
dotnet ef database update \
  --project "Wayd.Infrastructure/src/Wayd.Infrastructure.Migrators.MSSQL" \
  --startup-project "Wayd.Web/src/Wayd.Web.Api"
```

## Run the Application

### Option 1: .NET Aspire (Recommended)

```bash
cd Wayd.AppHost
dotnet run
```

Access the Aspire dashboard at `http://localhost:15888` for telemetry, logs, and service URLs.

### Option 2: Run Services Individually

```bash
# Terminal 1 - API
cd Wayd.Web/src/Wayd.Web.Api
dotnet run

# Terminal 2 - Frontend
cd Wayd.Web/src/wayd.web.reactclient
npm run dev
```

### Option 3: Docker Compose

```bash
docker compose up
```

Access points:
- API: `https://localhost:5001`
- Client: `http://localhost:5002`
- Seq (logs): `http://localhost:8081`
