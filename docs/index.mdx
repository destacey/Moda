# Welcome to Moda

Moda is a work management system used to plan, manage, and create associations across work items, projects, teams, planning, and products. It helps organizations track, align, and deliver work at scale.

## What is Moda?

Moda provides a unified platform for organizations to:

- **Organize** — Structure teams into hierarchies of teams and teams of teams, track operating models and team memberships over time
- **Plan** — Define planning intervals, increments, and sprints; manage objectives and risks across teams
- **Manage Work** — Track work items across workspaces with configurable work processes, workflows, and work types
- **Manage Portfolios** — Organize projects into portfolios and programs, track lifecycle status and strategic alignment
- **Align Strategy** — Define strategic themes that guide investment decisions and work prioritization
- **Integrate** — Synchronize data from Azure DevOps and Microsoft Entra ID

## Ownership Model

Moda is built to allow data from external systems to be synchronized into the platform. Objects in Moda that can come from external systems have an **Ownership** attribute:

- **Owned** — The object is created and managed within Moda
- **Managed** — The object is owned by an external system and is read-only in Moda

Managed objects can still be associated or linked to owned items. For example, a work item owned by Azure DevOps (managed in Moda) can be linked to a team objective owned by Moda.
