# Welcome to Moda

Moda is a work management system used to plan, manage, and create associations across work items, projects, teams, planning, and products. It helps track, align, and deliver work across organizations.

## What is Moda?

Moda provides a unified platform for organizations to:

- **Manage Work** - Track work items, bugs, user stories, and tasks across teams
- **Plan Strategically** - Define objectives, key results, and strategic initiatives
- **Organize Teams** - Structure teams, employees, and organizational hierarchies
- **Portfolio Management** - Manage projects and portfolios with full lifecycle tracking
- **Integrate External Systems** - Synchronize data from Azure DevOps and other platforms

## Ownership Model

Moda allows data from external systems to be synchronized into the platform. Objects in Moda have an **Ownership** attribute:

- **Owned** - The object is owned and managed within Moda
- **Managed** - The object is owned by an external system and is read-only in Moda

Managed objects can still be associated or linked to owned items. For example, a work item owned by Azure DevOps (managed in Moda) can be linked to a Team Objective owned by Moda.
